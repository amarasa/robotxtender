A plugin that allows you to overwriting your existing Robots.txt and automatically includes the sitemap

== Description ==

This plugin is designed to easily overwriting your Robots.txt and automatically insert in the sitemap, which is easily forgotten.


== Frequently Asked Questions ==

= How do I automatically insert in the sitemap? =

In the plugin settings, check the box that says to automatically include the sitemap.

= Do I need to know the URL to the sitemap? =

No, the sitemap will be dynamically pulled in through a WordPress function
